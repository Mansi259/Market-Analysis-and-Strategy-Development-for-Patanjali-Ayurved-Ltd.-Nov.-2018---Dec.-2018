# Market-Analysis-and-Strategy-Development-for-Patanjali-Ayurved-Ltd.-Nov.-2018---Dec.-2018
## 📌 Project Overview
This project analyzes the market position of Patanjali Ayurved Ltd. and proposes strategies to enhance its market share and customer engagement.

## 🎯 Objective
- Evaluate market position
- Analyze competitors
- Study consumer behavior
- Provide strategic recommendations

## 🛠 Methodology
- Market research
- Competitor analysis
- Consumer behavior study
- SWOT analysis
- Data visualization

## Tools Used
- Microsoft Excel
- Python (Pandas, Matplotlib)
- PowerPoint

## 📊 Key Insights
- Strong brand in Ayurvedic segment
- Competitive pricing
- Weak digital presence
- Growing demand for organic products

## 📈 Recommendations
- Improve digital marketing
- Expand premium product line
- Enhance branding
- Target younger audience
